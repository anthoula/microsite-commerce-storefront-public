The quickest path to a new storefront starts with our [Commerce boilerplate](https://github.com/hlxsites/boilerplate-commerce-dropins). So let's use it to create a starter storefront using Edge Delivery Services and Adobe Commerce dropin components built-in.

## Prerequisites

Before you get started, make sure you have the following accounts and tools set up on your local computer.

<Aside type="note" title="Choose One">

  only need an account for one of them, not both.
</Aside>

## Step-by-step

After completing the steps below, you will have a local development environment where you can learn about the anatomy of a dropin ([Explore dropins](/quick-start/explore-dropins/)) and find the resources available ([Find dropin resources](/quick-start/find-resources/)) to begin your Commerce storefront journey.

<Tutorial>

### Create your storefront repository

1. Navigate to [aem-boilerplate-commerce](https://github.com/hlxsites/aem-boilerplate-commerce).
1. Click the **Use this template** button and select **Create a new repository**.
1. Choose an owner and repository name.
1. (Optional) Choose a description and visibility, then click the **Create repository** button.

### Clone your storefront repository

1. Navigate to your new repository and click the **Code** button.
1. Copy the URL (HTTPS or SSH).
1. Open your terminal and navigate to the directory where you want to clone your storefront repository.
1. Run the following command, replacing `URL` with the URL you copied.

   ```bash frame="none"
   git clone URL
   ```

### Install the storefront dependencies

1. Navigate to your new storefront directory and run the following command:

   ```bash frame="none"
   npm i
   ```

### Add the AEM Code Sync app

1. Navigate to the [AEM Code Sync app](https://github.com/apps/aem-code-sync) and click the **Configure** button.
1. Select the organization account to add the app to.
1. From GitHub's AEM Code Sync page, in the **Repository access** section, select the repository you created and click the **Save** button.
1. If the installation was successful, you'll be redirected to a success screen on https://aem.live. You can close this browser tab.

### Start the local development server

1. Run the following command:

   ```bash frame="none"
   npm start
   ```

1. You should see the first page of your boilerplate storefront at `http://localhost:3000`.

### Click around your starter storefront!

You should see a basic storefront with a header, footer, and a few dropins.

</Tutorial>

<Aside type="note" title="Next steps">
