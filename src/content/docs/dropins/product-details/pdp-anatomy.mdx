The `ProductDetails` component is designed for rendering detailed information about a product in an e-commerce setting. It leverages the Preact library and is compatible with Adobe's Elsie UI framework. This component dynamically displays product details, including SKUs, pricing, descriptions, and customizable options, with support for internationalization and accessibility features.

## Props

The component accepts the following props:

- `sku`: (**Required**) The unique identifier for the product.
- `hideSku`: Optional boolean to hide the SKU display.
- `productData`: Optional initial product data to preload the component.
- `hideQuantity`: Hides the quantity selector if set to true.
- `hideShortDescription`, `hideDescription`, `hideAttributes`: Control visibility of respective product information sections.
- `hideURLParams`: Disables synchronization of options with URL parameters if true.
- `slots`: An object containing slot overrides for customizing various parts of the component display.
- `carousel`: Configuration for the product image carousel display.
- `onAddToCart`: Callback function triggered upon adding the product to the cart.

### `CarouselConfig` Object

- `mobile`: Optional boolean to optimize carousel for mobile devices.
- `controls`: Determines the type of navigation controls for the carousel (`'thumbnailsRow'`, `'thumbnailsColumn'`, `'dots'`, or `null`).

### Slot Configuration

Slots allow for the customization of component parts. Each slot accepts a `SlotProps` object with a context containing `data`, `values`, and a `valid` state. Custom actions and content can be injected through these slots.

## Usage Example

```tsx
<ProductDetails
  sku="12345"
  onAddToCart={(values) => console.log('Added to cart', values)}
  slots={{
    Title: {
      /* SlotProps configuration */
    },
    SKU: {
      /* SlotProps configuration */
    },
    // Define additional slots as needed
  }}
/>
```

## Callbacks and Event Handling

### onAddToCart(values: Values)

Triggered when the Add to Cart button is clicked. Receives an object with sku, quantity, and optionally optionsUIDs reflecting the user's selection.

## External Dependencies

This component relies on several external packages and Adobe Elsie UI components for functionality and styling:

- @adobe/elsie/lib and @adobe/elsie/components for foundational components and utilities.
- @adobe/elsie/icons for SVG icons.
- @adobe/event-bus for global event handling.
- @adobe/elsie/i18n for internationalization support.
- API functions (`getProductData`, `getRefinedProduct`) for fetching product details.
