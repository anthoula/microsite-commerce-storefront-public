The components used to build the PDP dropin are designed to be flexible and customizable.

## Carousel

UI component that displays a series of images in a rotating manner. Users can interact with it through navigation buttons or touch gestures to move between the items. Carousels are commonly used to showcase products, highlight information, or create slideshows on websites. They're favored for their space-saving design and interactive presentation of content.

### Props

| Name     | Type   | Default | Description                                                                                         |
| -------- | ------ | ------- | --------------------------------------------------------------------------------------------------- |
| children | Array  | []      | Array of images to be displayed in the carousel.                                                    |
| show     | number | 1       | Number the slides visible at a time per breakpoint. You may also pass a number for all breakpoints. |
