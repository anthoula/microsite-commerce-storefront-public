Learn about the slots provided in the Cart dropin.

## CartSummaryList SlotProps

The slots for Cart Summary List are defined in the `CartSummaryListProps` interface. Each slot comes with a set of [primitive functions](/dropins/all/extending/) and may be extended with custom functions.

```typescript
slots?: {
  Heading?: SlotProps;
  EmptyCart?: SlotProps;
  Footer?: SlotProps;
};
```

## Example customizations

Here are some examples of how you can customize the Cart dropin using the slots:

### Footer

Adding a promotional cart rule to each product in the Cart.

```js
return provider.render(Cart, {
  slots: {
    ProductList: (ctx) => {
      const productList = document.createElement('div');

      // Initial render of ProductList
      provider.render(CartSummaryList, {
        slots: {
          Footer: (ctx) => {
            // Prepend Product Promotions
            const discountLabel = ctx.item?.discount?.label;
            if (discountLabel) {
              discountLabel.map((label) => {
                const discount = document.createElement('div');
                discount.style.color = '#3d3d3d';
                discount.innerText = label;
                ctx.appendChild(discount);
              });
            }
          },
        },
      })(productList);

      ctx.replaceWith(productList);
    },
  },
})(document.getElementById('@dropins/cart'));
```
