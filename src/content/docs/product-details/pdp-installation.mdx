DropIns are designed to be used either at run-time through an ESM CDN or during build-time using NPM.

## Run-time (CDN)

```js
// Dropin Initializer
import { initializers } from 'https://unpkg.com/@dropins/tools/initializer.js';

// PDP DropIn
import * as pdp from 'https://unpkg.com/@dropins/storefront-pdp/api.js';
import { render as pdpProvider } from 'https://unpkg.com/@dropins/storefront-pdp/render.js';
import ProductDetails from 'https://unpkg.com/@dropins/storefront-pdp/containers/ProductDetails.js';
```

## Build-time (NPM)

If you use a bundler like Webpack or Vite, you can install the package from the npm registry as follows:

```bash
npm install @dropins/storefront-pdp
```

```js
// Dropin Initializer
import { initializers } from '@dropins/tools/initializer.js';

// PDP DropIn
import * as pdp from '@dropins/storefront-pdp/api.js';
import { render as pdpProvider } from '@dropins/storefront-pdp/render.js';
import ProductDetails from '@dropins/storefront-pdp/containers/ProductDetails.js';
```

## Usage 

### Dropin Initializer

```js
// Set endpoint configuration
pdp.setEndpoint('https://<catalog-service-enpoint>/graphql');

pdp.setFetchGraphQlHeaders({
  //... environment required headers
});

// Register PDP
initializers.register(pdp.initialize);

// Mount Initializers (must be called after all initializers are registered)
window.addEventListener('load', initializers.mount);
```

### Localization

Your language packs will merge with, and override entries from the PDP dropin, and Elsie SDK.

```js
// Language Packs
const en_US = await fetch('/i18n/en_US.json').then((res) => res.json());
const fr_FR = await fetch('/i18n/fr_FR.json').then((res) => res.json());

// Register PDP with language definitions
const langDefinitions = {
  default: en_US,
  en_US,
  fr_FR,
};

// Register Initializers
initializers.register(api.initialize, { langDefinitions });
```

### Render container

```js
// Render Containers
pdpProvider.render(ProductDetails, {
  sku: 'VA11-GO-NA', // <- Product SKU
  onAddToCart: (product) => {
    // custom add to cart logic
  },
})(document.getElementById('pdp')); // <- target DOM element
```

### Actions

```js
// Render Containers
await provider.render(ProductDetails, {
  sku: 'VA11-GO-NA',
  slots: {
    Actions: (ctx) => {
      // Add to Cart Button
      ctx.appendButton((next) => ({
        text: next.dictionary.PDP.Product.AddToCart.label,
        icon: 'Cart',
        disabled: !next.data?.inStock,
        onClick: () => console.log('Add To Cart', next.values),
      }));

      // Add to Wishlist Button
      ctx.appendButton((next) => ({
        text: next.dictionary.CustomButtons.AddToWishlist.label,
        icon: 'Heart',
        variant: 'secondary',
        onClick: () => console.log('Add To Wishlist', next.values),
      }));

      // Add custom element
      const link = document.createElement('a');
      link.href = '#';
      link.innerText = 'Some Link';
      ctx.appendHTMLElement((next) => {
        link.innerText = next.dictionary.Custom.someLink.label;
        return link;
      });
    },
  },
})(document.getElementById('pdp'));
```
