## Create a new file

Create a new file called `brand-drop-ins.css` in your project.

## Override the Base design tokens

Override the Base design tokens to match your brand. The example below shows how to override the Base design tokens to match your brand.

```css
:root {
  --color-primary: #ff0000;
  --color-secondary: #00ff00;
  --color-tertiary: #0000ff;
  --color-text: #000000;
  --color-background: #ffffff;
  --color-border: #000000;
}
```

## Import the brand-drop-ins.css file

Import the `brand-drop-ins.css` file into your project.

```html
<link rel="stylesheet" href="brand-drop-ins.css" />
```
